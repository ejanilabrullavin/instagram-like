# instagram-like

Simple instagram-like example

## Features
- Feature 1: User authentication
- Feature 2: Photo sharing

## Installation
To install the project, run:

```
npm install instagram-like
```

## Usage
To use the project:

```
import InstagramLike from 'instagram-like';
```
